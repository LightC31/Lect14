# SVG Проєкт

Цей проєкт демонструє роботу з **SVG** та **CSS-анімацією**.  
Використано чотири прості фігури: квадрат, прямокутник, коло та еліпс, кожна з унікальними кольорами та товщиною контуру. 


## Використані технології
- HTML5  
- CSS3  
- SVG  

## Функціонал та анімації
- **Коло**: постійно змінює кольори заливки та контуру.  
- **Еліпс**: пульсує та змінює кольори.  
- **Квадрат**: змінює кольори при наведенні миші.  
- **Прямокутник**: змінює кольори при наведенні миші.  

## Як запустити проєкт локально
1. Клонувати або завантажити репозиторій.  
2. Відкрити файл **index.html** у будь-якому сучасному браузері.  



## Публікація на GitHub Pages
1. Відкрити **Settings → Pages** у репозиторії.  
2. В полі *Source* обрати **main branch / root**.  
3. Натиснути **Save**.
